<h1>Silhouette &mdash; Token Forge</h1>
<p>A powerful in-browser token creation tool for <strong>Foundry VTT v13</strong></p>
<p>Create professional tokens directly inside Foundry VTT &mdash; no Photoshop needed. Load a character portrait, remove the background with magic wand &amp; brushes, add a metallic token ring, and export a polished PNG in seconds.</p>
<hr />
<h2>&nbsp;Features</h2>
<h3>&nbsp;Background Removal</h3>
<ul>
<li><strong>Magic Wand</strong> &mdash; Click to select similar-colored areas with adjustable tolerance</li>
<li><strong>Marquee / Lasso</strong> &mdash; Rectangular and freeform selection</li>
<li><strong>Eraser</strong> &mdash; Pixel-level cleanup with adjustable brush size</li>
<li><strong>Restore Brush</strong> &mdash; Bring back over-erased pixels from the original image</li>
</ul>
<h3>Token Ring System</h3>
<ul>
<li><strong>10 built-in SVG token rings</strong> with realistic metallic textures (gold, silver, crimson, arcane blue, etc.)</li>
<li><strong>Load custom ring images</strong> &mdash; Use your own PNG/SVG ring designs</li>
